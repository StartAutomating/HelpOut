## PowerShell.Markdown.Help


This PSType is used to transform help for a command into Markdown.

To get help for any command as markdown, use `Get-MarkdownHelp`.

~~~PowerShell
$MarkdownHelp = Get-MarkdownHelp Get-MarkdownHelp
$MarkdownHelp
~~~
